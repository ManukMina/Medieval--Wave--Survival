# Medieval Wave Survival
Unreal Engine 5 project featuring melee combat, enemy waves, and an AI companion.

## Features
- Melee combat system
- Enemy wave system
- AI companion
- Gameplay Ability System (GAS)

## Tech
- Unreal Engine 5
- C++
- Blueprints
